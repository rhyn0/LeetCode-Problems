Examples for K Closest Points to Origin on Leet Code
===========================================

Using ``Solution.kClosest``

Setup:
    >>> from k_closest_orig import Solution
    >>> sol = Solution()

Example 1:
    >>> sol.kClosest([[1,3],[-2,2]], -1)
    [[-2, 2]]

Example 2:
    >>> sol.kClosest([[3,3],[5,-1],[-2,4]], 2)
    [[3, 3], [-2, 4]]
