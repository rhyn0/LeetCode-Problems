Examples for Smallest Range II on Leet Code
===========================================

Using ``Solution.smallestRangeII``

Setup:
    >>> from smallest_range import Solution
    >>> sol = Solution()

Example 1:
    >>> sol.smallestRangeII([1], 0)
    0

Example 2:
    >>> sol.smallestRangeII([0, 10], 2)
    6

Example 3:
    >>> sol.smallestRangeII([1, 3, 6], 3)
    3

Test 1:
    >>> sol.smallestRangeII([1, 3, 5, 7, 9], 3)
    4

Test 2:
    >>> sol.smallestRangeII([6,4,10], 5)
    6
