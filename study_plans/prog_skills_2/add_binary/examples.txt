Examples for Add Binary on Leet Code
===========================================

Using ``Solution.addBinary``

Setup:
    >>> from add_binary import Solution
    >>> sol = Solution()

Example 1:
    >>> sol.addBinary("11", "1")
    '100'

Example 2:
    >>> sol.addBinary("1010", "1011")
    '10101'

Example 3:
    >>> sol.addBinary("0", "0")
    '0'
