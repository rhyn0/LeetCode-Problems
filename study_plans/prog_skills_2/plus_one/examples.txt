Examples for Plus One on Leet Code
===========================================

Using ``Solution.plusOne``

Setup:
    >>> from plus_one import Solution
    >>> sol = Solution()

Example 1:
    >>> sol.plusOne([1,2,3])
    [1, 2, 4]

Example 2:
    >>> sol.plusOne([4,3,2,1])
    [4, 3, 2, 2]

Example 3:
    >>> sol.plusOne([9])
    [1, 0]
