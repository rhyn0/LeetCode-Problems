Examples for Monotonic array on Leet Code
=========================================

Using ``Solution.isMonotonic``

Setup:
    >>> from monotonic_array import Solution
    >>> sol = Solution()


Example 1:
    >>> sol.isMonotonic([1, 2, 2, 3])
    True

Example 2:
    >>> sol.isMonotonic([6, 5, 4, 4])
    True

Example 3:
    >>> sol.isMonotonic([1, 3, 2])
    False

Example 4:
    >>> sol.isMonotonic([1])
    True
