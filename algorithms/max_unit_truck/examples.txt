Examples for Maximum Units on Truck on Leet Code
===========================================

Using ``Solution.maximumUnits``

Setup:
    >>> from max_unit_truck import Solution
    >>> sol = Solution()

Example 1:
    >>> sol.maximumUnits([[1, 3], [2, 2], [3, 1]], truckSize=4)
    8

Example 2:
    >>> sol.maximumUnits([[5, 10], [2, 5], [4, 7], [3, 9]], truckSize=10)
    91
