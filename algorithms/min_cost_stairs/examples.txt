Examples for Min Cost Climbing Stairs on Leet Code
===========================================

Using ``Solution.minCostClimbingStairs``

Setup:
    >>> from min_cost_stairs import Solution
    >>> sol = Solution()

Example 1:
    >>> sol.minCostClimbingStairs([10, 15, 20])
    15

Example 2:
    >>> sol.minCostClimbingStairs([1, 100, 1, 1, 1, 100, 1, 1, 100, 1])
    6

Test 1:
    >>> sol.minCostClimbingStairs([1, 100])
    1
